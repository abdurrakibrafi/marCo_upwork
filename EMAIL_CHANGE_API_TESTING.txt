# Email Change API Testing Flow

## 1. Login (Get Auth Token)
**POST** `http://localhost:8000/api/auth/login/`
```json
{
  "email": "your-email@example.com",
  "password": "your-password"
}
```
**Response:** Save the `access` token for auth

---

## 2. Initiate Email Change (Send OTP to New Email)
**POST** `http://localhost:8000/api/auth/profile/change-email/`
**Headers:** `Authorization: Bearer {access_token}`
```json
{
  "new_email": "new-email@example.com"
}
```

---

## 3. Verify Email Change (Use OTP)
**POST** `http://localhost:8000/api/auth/profile/verify-email-change/`
**Headers:** `Authorization: Bearer {access_token}`
```json
{
  "otp": "1234"
}
```

---

## 4. Resend Email Change OTP
**POST** `http://localhost:8000/api/auth/profile/resend-email-change-otp/`
**Headers:** `Authorization: Bearer {access_token}`
```json
{
  "new_email": "new-email@example.com"
}
```

---

## 5. Cancel Email Change
**POST** `http://localhost:8000/api/auth/profile/cancel-email-change/`
**Headers:** `Authorization: Bearer {access_token}`
```json
{
  "new_email": "new-email@example.com"
}
```

---

## Test Cases

### Invalid OTP
**POST** `http://localhost:8000/api/auth/profile/verify-email-change/`
```json
{
  "otp": "0000"
}
```
**Expected:** 400 Bad Request

### Email Already Taken
**POST** `http://localhost:8000/api/auth/profile/change-email/`
```json
{
  "new_email": "existing-user@example.com"
}
```
**Expected:** 400 Bad Request

### Missing Auth Token
**POST** `http://localhost:8000/api/auth/profile/verify-email-change/`
**No Authorization header**
```json
{
  "otp": "1234"
}
```
**Expected:** 401 Unauthorized
